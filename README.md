# lyrics-search

A clean, fast web application for searching song lyrics built entirely in vanilla JavaScript.

## Features

- **Search lyrics** - Find lyrics for your favorite songs quickly
- **Pure JavaScript** - No frameworks or dependencies required
- **Fully responsive** - Works seamlessly on desktop, tablet, and mobile devices
- **Open source** - Free to use and modify for your own projects

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start searching for lyrics!
